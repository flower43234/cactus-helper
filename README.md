# 🌵 CACTUS CLEANER v1.0

Привет! Это **Кактус** — легкий и дерзкий скрипт для тех, кому надоело, что Windows живет своей жизнью. Я собрал всё самое нужное в один `.bat` файл: чистка мусора, разгон для игр и полный контроль над тем, что запускается у тебя в системе.

Никаких тяжелых установщиков и лишней графики. Только текст и реальная польза.

---

### 📂 Описание разделов: на что мы жмем?

* **[1] FULL CLEAN (Очистка)** — «Генеральная уборка». Сносит временные файлы системы (`Temp`) и кэш `Prefetch`. Безопасно освобождает место, которое Windows копит месяцами.
* **[2] REGISTRY RESET (Реестр)** — «Кнопка спасения». Если вирус или кривой софт подменили системные файлы, этот пункт возвращает стандартный рабочий стол (`explorer.exe`) на место. Помогает при «черном экране».
* **[3] TOTAL STARTUP MANAGER (Автозагрузка)** — «Детектор нахлебников». Самая мощная часть. Видит всё: и обычные проги, и хитрые задачи в Планировщике. Ты просто вводишь номер программы — и она больше не запускается без твоего ведома.
* **[4] ADVANCED TWEAKS (Приватность и DNS)** — Отключает «слежку» Windows (телеметрию), усыпляет Кортану и ставит быстрый DNS от Cloudflare (1.1.1.1). Интернет станет стабильнее, а система — тише.
* **[5] SYSTEM OPTIMIZE (OneDrive и Мощь)** — «Тяжелая артиллерия». Полностью вырезает OneDrive из системы и включает скрытый режим «Максимальная производительность». 
    * *Примечание:* Кулеры могут начать шуметь сильнее — это нормально, значит процессор готов выдавать максимум частоты.
* **[6] TURBO MODE (Отклик)** — Делает интерфейс Windows резким. Убирает лишние задержки при открытии меню и окон.

### 🛠 Как пользоваться

1.  Скачай файл `cactus.bat`.
2.  Нажми на него правой кнопкой — **Запуск от имени администратора** (это обязательно, чтобы Кактус мог менять системные настройки).
3.  В меню просто нажимай нужные цифры. Я настроил мгновенный отклик: для большинства пунктов даже Enter нажимать не надо.
4.  Выход или возврат в главное меню — всегда кнопка **0**.
